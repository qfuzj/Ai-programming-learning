# 二、业务模块范围

后端需覆盖以下核心模块：

1. 用户认证与授权模块

2. 景点信息管理模块
- 管理员对景点进行增删改查
- 多条件筛选：地区、分类、等级、评分等

- 浏览历史查看
- 管理员审核点评内容（通过/拒绝/隐藏）

4. 行程计划模块
- 用户创建行程计划：目的地、日期、预算、同行人类型
- 行程按天添加景点/餐饮/住宿安排项
- 支持手动创建
- 支持 LLM 辅助生成
- 用户个人中心管理行程计划

5. AI智能推荐模块

- 景点详情页相似推荐
- 推荐结果与效果追踪数据落库（点击、收藏）

6. AI对话助手模块

- 对话历史持久化
- 用户查看历史会话

7. 运营管理后台模块
- 地区数据管理（省/市/区三级）
- 标签管理
- 用户账号管理（禁用/启用）
- 系统配置管理
- 操作日志查看
- ECharts 可视化看板：
  - 用户增长趋势
  - 景点热度排行

  - LLM 调用量与成本统计

8. 文件与基础设施模块
