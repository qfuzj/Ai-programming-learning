
# 前端页面与路由设计

## 1. 文档说明

### 1.1 前端技术栈
- Vue 3
- Vite
- Naive UI
- Pinia
- Vue Router
- vicons

### 1.2 设计目标
- 用户前台：突出沉浸式旅游内容展示、推荐与AI交互
- 管理后台：突出信息密度、数据管理效率、运营分析
- 页面结构需支持后续快速对接后端 RESTful API

## 3. 页面架构设计

### 3.1 布局划分
- `UserLayout`：用户前台主布局，含顶部导航、搜索栏、页脚
- `AdminLayout`：后台主布局，含侧边菜单、顶部导航、面包屑、内容区域
- `BlankLayout`：登录、404、403等空白页布局

### 3.2 页面风格建议
#### 用户前台
- 首页 Hero Banner + 搜索框 + 推荐卡片区
- 大图轮播与卡片流结合
- AI入口按钮高曝光
- 景点详情页强调图文沉浸感、相关推荐与评论内容

#### 管理后台
- 左侧菜单 + 顶部统计
- Naive UI 表格、表单、抽屉、弹窗为主
- 看板页使用卡片+图表布局
- 复杂列表配合筛选栏与分页器
