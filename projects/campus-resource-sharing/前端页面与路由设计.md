# campus-resource-sharing 前端页面与路由设计

## 1. 文档说明

本文档用于规范前端页面结构、路由配置、页面职责和模块划分，适用于 Vue3 + Vite + Naive UI + Pinia 项目开发。

---

## 2. 前端技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- Naive UI
- Axios
- @vicons
- 可选：unplugin-auto-import、unplugin-vue-components

---

## 3. 主题设计规范

- 主色：`#ffe60f`
- 主文字色：`#000000`
- 页面风格：简洁、清新、校园化
- 商品展示风格：卡片式布局
- 管理后台：左侧菜单 + 顶部栏 + 内容区

---

## 7. 路由守卫设计

## 7.1 用户端守卫规则
- 未登录不能访问：
    - `/publish`
    - `/me/**`
- 未登录时跳转 `/login`

## 7.2 管理端守卫规则
- 未登录管理员不可访问 `/admin/**`
- 非管理员登录不可访问后台页面
- 无权限时跳转 `/admin/login`

## 11. AI 前端代码生成要求

后续 AI 生成前端代码时必须遵守：
1. 路由结构严格按本文档设计
2. 页面文件路径尽量保持一致
3. 统一使用 Axios 请求封装
4. 统一使用 Pinia 管理用户状态
5. 用户端与管理端布局分离
6. 页面风格必须体现主色 `#ffe60f`
7. 表单使用 Naive UI 校验
8. 商品、订单、收藏、消息相关页面优先生成
